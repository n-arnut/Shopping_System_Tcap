ส่วนของ Frontend (Next.js Shopping System)  
 1.  ติดตั้งแพ็กเกจที่จำเป็น: เปิดเทอร์มินัลและรันคำสั่ง "npm install"
 2.  เริ่มต้นเซิร์ฟเวอร์สำหรับพัฒนา: หลังจากติดตั้งเสร็จสิ้น ให้รันคำสั่ง "npm run dev"
